# PromptWars: Indore — Event Rules & Info

**Event:** PromptWars (In-person) — Indore
**Organized by:** Google for Developers, H2S (Hack2Skill), Build with AI

---

## 1. The Social Contest

Your code isn't the only thing that should be trending today. Capture the vibe, share the grind, and claim your status.

### The Challenge
Create a social post (LinkedIn) about your `#PromptWars` experience at PromptWars: Indore.

### The Prize
The participant with the highest engagement (**Likes + Comments + Shares**) by the end of the event will be officially awarded.

### What to Include
- A photo of your setup or the venue.
- Use `#BuildwithAI` `#PromptWars`
- Tag `@googlefordevelopers` `@hack2skill`

---

## 2. Evaluation Framework

Each phase will be evaluated based on the **Code Assessment (Platform Scoring)**.

### Code Assessment Signals
Code assessment is performed automatically on the platform. It checks signals such as:
- Code Quality
- Security
- Efficiency
- Testing
- Accessibility
- Problem Statement Alignment

### Scoring Rules
- Code assessment scores contribute to the leaderboard **only for the Challenge**. Warm-up scores do **not** count.
- Only the **final submission score** for that phase is considered. The **best attempt score is not used**.
